# دليل الاستخدام السريع - مساعد الأنظمة السعودية الذكي

## 🎯 نظرة عامة

هذا النظام عبارة عن مساعد ذكي يستخدم الذكاء الاصطناعي للإجابة على الأسئلة المتعلقة بالأنظمة واللوائح في المملكة العربية السعودية.

## ✨ المميزات

- **استخراج تلقائي للأنظمة**: يقوم بجمع جميع الأنظمة من الموقع الرسمي تلقائياً
- **إجابات ذكية**: يستخدم الذكاء الاصطناعي المحسّن للغة العربية
- **بحث دلالي**: يفهم السياق والمعنى وليس فقط الكلمات المطابقة
- **مراجع كاملة**: كل إجابة تتضمن اسم النظام ورقم المادة وتاريخ النشر والرابط
- **تحديث يومي**: يتحقق تلقائياً من أي تغييرات في الأنظمة
- **واجهة ويب سهلة**: واجهة جميلة وسهلة الاستخدام باللغتين العربية والإنجليزية

## 🚀 خطوات الإعداد السريع

### 1. تثبيت المتطلبات

```bash
# تشغيل سكريبت الإعداد
bash setup.sh

# أو يدوياً
pip install -r requirements.txt
```

### 2. إضافة مفاتيح API

قم بتحديث ملف `.env` بالمفاتيح الخاصة بك:

```
SUPABASE_URL=https://exilgibjcnrtashitzjk.supabase.co
SUPABASE_SERVICE_ROLE_KEY=المفتاح_الخاص_بك
OPENAI_API_KEY=مفتاح_OpenAI_الخاص_بك
```

للحصول على مفتاح OpenAI:
1. قم بزيارة https://platform.openai.com/api-keys
2. سجل الدخول أو أنشئ حساب جديد
3. أنشئ مفتاح API جديد
4. انسخه وضعه في ملف `.env`

### 3. جمع الأنظمة من الموقع

```bash
# جمع كامل للأنظمة (المرة الأولى)
python saudi_law_scraper.py
```

هذا الأمر سيقوم بـ:
- جمع جميع مجلدات الأنظمة
- استخراج كل نظام من كل مجلد
- تحليل المواد من كل نظام
- حفظ كل شيء في قاعدة البيانات

### 4. إنشاء الفهارس الذكية

بعد جمع الأنظمة، قم بإنشاء الفهارس الذكية (embeddings):

```bash
curl -X POST https://exilgibjcnrtashitzjk.supabase.co/functions/v1/generate-embeddings \
  -H "Authorization: Bearer المفتاح_العام_لـSupabase"
```

### 5. تجربة النظام

افتح الملف `web/index.html` في متصفحك!

## 💡 أمثلة على الأسئلة

جرب هذه الأسئلة:

- ما هي حقوق الموظف عند إنهاء العقد؟
- ماذا ينص نظام العمل عن ساعات العمل الإضافية؟
- ما هي شروط تأسيس شركة في السعودية؟
- ما هي حقوق المستهلك في نظام التجارة الإلكترونية؟
- كيف يتم احتساب إجازة الموظف السنوية؟

## 🔄 التحديث اليومي

لفحص التحديثات في الأنظمة:

```bash
# فحص التحديثات
python saudi_law_scraper.py update
```

### جدولة التحديث التلقائي

لتشغيل التحديث تلقائياً كل يوم:

```bash
# إضافة إلى crontab (يعمل يومياً الساعة 2 صباحاً)
crontab -e

# أضف هذا السطر:
0 2 * * * cd /المسار/للمشروع && python saudi_law_scraper.py update
```

## 📱 استخدام واجهة الويب

1. افتح `web/index.html` في المتصفح
2. اكتب سؤالك في مربع البحث
3. اختر اللغة (عربي أو إنجليزي)
4. اضغط على "بحث"
5. ستظهر الإجابة مع جميع المراجع القانونية

## 🔌 استخدام API مباشرة

### إرسال سؤال

```bash
curl -X POST https://exilgibjcnrtashitzjk.supabase.co/functions/v1/query-law \
  -H "Authorization: Bearer المفتاح_العام" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "ما هي حقوق الموظف؟",
    "language": "ar",
    "max_results": 5
  }'
```

### الرد

```json
{
  "answer": "بناءً على نظام العمل السعودي...",
  "citations": [
    {
      "law_name_ar": "نظام العمل",
      "law_number": "م/51",
      "article_number": "74",
      "publication_date": "2005-04-23",
      "law_url": "https://laws.boe.gov.sa/..."
    }
  ],
  "response_time_ms": 1250
}
```

## 📊 صيغة الإجابات

كل إجابة من الذكاء الاصطناعي تتضمن:

1. ✅ **اسم النظام** - الاسم الكامل للنظام
2. ✅ **رقم المادة** - رقم المادة المحددة
3. ✅ **تاريخ النشر** - تاريخ نشر النظام
4. ✅ **رابط النظام** - رابط مباشر للنظام على الموقع الرسمي

## 🛠️ حل المشاكل الشائعة

### لا توجد نتائج

1. تأكد من أن الأنظمة تم جمعها: تحقق من جداول قاعدة البيانات
2. تأكد من إنشاء الفهارس الذكية
3. تحقق من مفتاح OpenAI

### البرنامج لا يجمع أي بيانات

1. قد تكون بنية الموقع تغيرت - افحص HTML يدوياً
2. حدّث CSS selectors في ملف `saudi_law_scraper.py`
3. تأكد من الوصول للموقع

### النص العربي يظهر بشكل خاطئ

1. تأكد من أن الترميز UTF-8 مفعل في كل مكان
2. تحقق من إعدادات قاعدة البيانات
3. تأكد من أن المتصفح يدعم العربية

## 📈 نصائح للأداء الأفضل

1. **المعالجة الدُفعية**: أنشئ الفهارس على دفعات لتجنب تجاوز الحد
2. **التخزين المؤقت**: احفظ الأسئلة الشائعة مؤقتاً
3. **الفهرسة**: تأكد من وجود فهارس مناسبة
4. **التباطؤ**: أضف تأخيرات بين طلبات الاستخراج
5. **المراقبة**: استخدم جدول `query_logs` لتتبع الأداء

## 🔐 ملاحظات أمنية

1. لا تكشف مفتاح service role في الكود العام
2. استخدم Row Level Security للإنتاج
3. تحقق من صحة جميع المدخلات
4. ضع حدود للمعدل لمنع الإساءة
5. احفظ مفتاح OpenAI بأمان

## 📞 الدعم

للمشاكل المتعلقة بـ:
- **Supabase**: تحقق من سجلات لوحة التحكم
